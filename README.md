# 개인정보처리방침 (Privacy Policy)

**기억해voca**(이하 '앱')는 관련 법령을 준수하며, 앱이 처리하는 정보와 이용 목적을 다음과 같이 안내합니다.

## 1. 서비스 구성

앱의 단어 학습 기능은 회원가입이나 로그인 없이 오프라인에서 사용할 수 있습니다. 리더보드는 사용자가 선택해 이용하는 온라인 기능입니다.

## 2. 처리하는 정보

### 단말기에만 저장되는 정보

- 사용자가 불러온 단어장 파일
- 학습 진도, 즐겨찾기, 완료 상태 및 학습 통계
- 리더보드 계정 복구용 6개 단어 문구: Android의 보호된 저장소에 저장되며 서버로 전송하지 않습니다.

이 정보들은 학습 기능 제공을 위해 앱의 내부 저장소에 보관됩니다. 사용자가 앱을 삭제하면 단말기에 저장된 정보도 함께 삭제됩니다.

### 리더보드 이용 시 서버에 저장되는 정보

- 사용자가 입력한 닉네임
- Firebase에서 발급한 계정 식별자
- 학습 온도, 연속 학습일, 누적 학습일
- 최종 갱신 시각

이 정보는 리더보드 프로필 생성, 순위 표시, 계정 복구 및 부정한 접근 방지를 위해 처리됩니다. 앱은 사용자의 실제 이메일 주소나 전화번호를 요구하지 않습니다. 닉네임에 실명 등 본인을 알아볼 수 있는 내용을 입력하지 마시기 바랍니다.

## 3. 자동으로 처리될 수 있는 정보

앱 및 제3자 서비스의 제공·보안·광고 기능을 위해 다음 정보가 자동으로 처리될 수 있습니다.

- 광고 식별자
- 기기 및 운영체제 정보
- IP 주소, 접속 시각, 앱 무결성·보안 확인에 필요한 기술 정보

## 4. 제3자 서비스

앱은 Google이 제공하는 다음 서비스를 사용합니다.

- **Firebase Authentication**: 리더보드 계정 인증
- **Cloud Firestore**: 리더보드 프로필과 순위 정보 저장
- **Firebase App Check**: 정상적인 앱에서 오는 요청인지 확인
- **Google AdMob**: 광고 제공 및 광고 효과 분석

제3자 서비스의 정보 처리에는 Google의 정책이 적용됩니다.

- [Google 개인정보처리방침](https://policies.google.com/privacy)
- [Google 광고 기술 정책](https://policies.google.com/technologies/ads)

## 5. 보유 및 파기

- 단말기의 단어장·학습 기록은 앱을 삭제할 때 함께 삭제됩니다.
- 리더보드 정보는 사용자가 앱의 **리더보드 계정 삭제** 기능을 실행할 때까지 보관됩니다.
- 앱만 삭제하면 서버의 리더보드 계정은 자동으로 삭제되지 않습니다. 앱 삭제 전에 계정을 삭제하거나, 복구 단어로 다시 복구한 뒤 삭제할 수 있습니다.

## 6. 이용자의 권리

이용자는 앱에서 복구 단어를 새로 발급하거나 리더보드 계정을 삭제할 수 있습니다. 앱을 삭제하면 단말기에 보관된 정보를 삭제할 수 있습니다. 광고 개인 설정은 Android 단말기의 개인정보 또는 광고 설정에서 변경할 수 있습니다.
